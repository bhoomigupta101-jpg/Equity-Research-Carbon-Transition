# Equity-Research-Carbon-Transition
A financial stress-testing model evaluating carbon tax impacts and transition capex requirements for the global steel industry.

# Global Steel Carbon Model.xlsx:
The primary financial engine. Includes dynamic scenario toggles, CBAM tax logic and the executive dashboard.

# Carbon Data Analysis.ipynb:
Python-based exploratory data analysis (EDA) used to validate emissions intensities and correlate carbon pricing with margin impact.

# Dashboard Preview.png:
High-level visual summary of the portfolio's carbon risk exposure and investment recommendations.

# .csv Files:
Processed data exports provided for quick browser-based viewing of model inputs and stress-test results.
